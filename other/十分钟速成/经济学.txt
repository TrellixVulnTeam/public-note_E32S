一、经济学
	经济学是研究人与决策的科学，更具体来说，研究人如何获得收入，如何花钱。
	# 两个假设：
		* 资源短缺，欲望无线
		* 所有事情都有成本
	 -> 从有限的资源中获得最大的利益
	# 经济学分类：
		* 宏观经济学
			研究经济整体，关注国家层面的经济情况，例如产出、失业、通货膨胀、政府开销和经济增长。
		* 微观经济学
			研究单个经济单位，例如个人、组织和企业的盈利策略。
	# 机会成本
		一切活动都是有成本的，选择了某项活动，就意味着放弃了其他活动。
		从众多选项中进行了选择后，其他选项的最大收益，就是机会成本。机会成本越大，选择就越纠结。
			
二、专业化与贸易
	# 《国家财富的本质与原因的研究》
		又名《国富论》。
		* 提出国家富有的原因是【专业化】，又叫做【劳动分工】。
		* 提出人们变得富有的原因是【贸易交换】
	# 生产可能性边界(Production Possibilities Frontier, PPF)
		反映出有效利用资源生产出的【两种】商品的不同组合形式。
		* 横纵坐标表示两种不同商品A和B的数量，每个国家资源有限，因此在坐标系统中都存在着一条各自不同的边界线，并无法生产出超出边界线的商品。
			若生产的AB商品数量的坐标点位于PPF内部，则资源没有得到充分利用，有剩余。
			若生产AB商品数量的坐标点位于PPF外部，则资源不够用，是无法这样生产的。
			若在PPF上，则资源被高效利用。
		* 不同的国家PPF是不同的
			国家α生产1个A，需要牺牲N个B。
			国家β生产1个A，需要牺牲M个B。
			若1个A通过贸易，可以换的P个B，并且通常有N<P<M。
			则说明α应该生产更多的A，并通过贸易换取B，这样牺牲相同的A可以增加B的个数。
			也说明β应该生产更多的B，并通过贸易换取A，这样牺牲相同的B可以增加A的个数。
			因此每个国家应该生产自己较擅长的商品，再通过贸易补齐不足。

三、经济体系与宏观经济
	# 生产要素
		生产要素指的是生产产品所需要的主要投入
		* 土地
		* 劳动力
		* 资本
	# 经济体系
		本质上是【谁，拥有和控制生产要素】的问题。
		* 计划经济
			【政府】控制生产要素。社会主义经济，指的是既有私有财产(私有制)，又有国有产业(公有制)。
			极端计划经济是朝鲜。
		* 市场经济
			【个体】拥有生产要素。资本主义经济，政府对生产、商务和贸易采取放任自由的态度。
			极端市场经济是新西兰，几乎没有税收和调控。
		* 现代经济
			自由市场和计划经济【共存】的经济体制。不同的国家政府对市场的调控力度不同。
			政府在现代经济中有非常重要的身份：
				1.维护法制。法律，警察，条款和法院。
				2.公关基础设施和服务。马路，桥梁，教育和国防。
				3.市场出现恶性问题，政府介入干预。例如垄断，工人待遇，对产品进行监管等。
			政府的钱 一部分来自于家庭和企业的税收，一部分来自于贷款。

四、供给与需求
	# 市场对价格的影响
		一个产品的在市场上的价格来源于供给与需求曲线。
		纵坐标是产品价格，横坐标是产品的数量
			需求曲线一般是价格越低，用户需求的产品越多。
			供给曲线一般是价格越低，商家供给的产品越少。
			当价格过高，商家生产的更多，用户需求更少，供大于求，产品【过剩】，将会导致价格下跌。
			当价格过低，商家生产的更少，用户需求更多，供不应求，产品【短缺】，将会导致价格上涨。
			价格一般会稳定于需求曲线和供给曲线的交点，但是由于季节，技术等问题，需求曲线一直在改变，因此价格通常不会完全稳定。
			
五、宏观经济
	宏观经济学研究整体经济，1930年大萧条时期，经济学家才感受到用系统方法衡量整体经济的需要。
	# 策略者的目标
		* 维持经济持续增长
			通过国内生产总值(Gross Domestic Product, GDP)衡量。
			GDP是一段时间(通常是一年)，一个国家境内所有最终产品和服务的价值，并以美元衡量。
			GDP并不包括经济体内的所有交易，
				股票，收购，非法交易，传统生产(自给自足，比如水管工自己给自己修水管)，不产生新产品和服务的等都不会计入GDP。
			其实美元也无法完全公正的衡量GDP，因为GDP代表的是价值，而美元是价格，不同的国家通货膨胀情况不同，因此有时GDP会进行通货膨胀矫正。
				比如，AB两个国家，生产相同多的汽车，但是A国卖的更贵(通货膨胀)，所以A国的GDP会更高，但其实GDP情况应该是相同的。
		* 控制失业
			通过失业率衡量。
			失业率 = 失业人口数目/劳动力人口数目 * 100%
				劳动力人口，表示处在合法劳动年龄 或 积极找工作的人(就是有工作需求的所有人)。
			失业的原因：
				1.摩擦性失业
					人们【暂时】没有工作，或是出于找工作期间。
				2.结构性失业
					市场不需要特定类型的劳动了，比如某些机器代替人类。
				3.周期性失业
					经济原因人们停止购买产品，工厂进行裁员，工人收入降低，更多的人停止消费，更多的人失业。陷入恶性循环。
			1和2两种失业是自然性失业，无法完全隔绝。主要是要降低第3点的失业。
		* 维持价格稳定
			通过通货膨胀率衡量。通过追踪一定量一般采购物品的价格(购物篮)，通货膨胀率随时间变化的百分比。
			通货膨胀，这会降低货币的购买能力，工人要求涨工资，企业成本增加，利率增加，贷款更难。
			通货紧缩，意味着GDP会下降，失业率会增加，严重的经济衰退意味着通货紧缩，人们对商品和服务的需求减少。
			
六、生产力与增长
	# 富有
		经济学家通过【人均GDP】来衡量一个国家的富有程度。
		人均GDP = GDP/国家人口，代表每个人的产出。
	生产力是人均GDP增长的根本原因。但是人均GDP增长也可能是贫富差距进一步拉大的结果。
	
七、通货膨胀与泡沫
	# 通货膨胀矫正
		经济学家挑选一系列产品来代表一般消费者每年会购买的东西。例如房租、汽油、面包和电影票等。再把下一年的价格相加，同样再计算下一年。
		【真实(Real)】指已经通货膨胀矫正的价格
		【名义(Nominal)】指没经过通货膨胀矫正的价格
	# 通货膨胀的原因
		富有，并不在于有多少钱，而是在于拥有多少购买力。
		* 需求拉动通货膨胀
			货币增多，商品没有变化，商品所需要的货币就变多。
		* 供给冲击
			重要资源的减少，会导致产业链上的商品变贵。本质上就是货币不变，商品减少，商品所需要的货币就变多。
	# 泡沫
		泡沫之所以会发生，主要的核心在于产品不是用来消费，而是用来挣钱。
		买方需求不断上涨，每个人都认为自己可以从中赚钱，促使更多的人进入购买，为了赚钱价格被太高。
		
八、财政政策
	财政政策用于矫正通货膨胀。手段是改变政府的支出和税收。
	# 经济学缺口
		衰退缺口，指的是产出大于需求，工人下岗，失业率增加。
		通货膨胀缺口，指的是产出小于需求，商人提高稀有资源价格，进而导致产业链上的产品价格增加，以此降低需求，物价上升。
	# 常用手段
		* 衰退缺口期间
			政府增加支出，或者减少税收。
			增加支出，可以提高工人，教师等岗位的收入，刺激他们进行消费，这增加了其他岗位的收入。
			减少税收，可以直接增加群众的收入。
			收入的增加，意味着需求的上涨，抑制衰退。
			政府增加支出，这笔钱由负债来出。也就是赤字支出，政府会去借钱。
		* 通货膨胀缺口期间
			政府减少支出，或者增加税收。
	# 负债
		为了在经济衰退时刺激经济复苏，政府增加支出，因此政府需要借钱。也叫做赤字支出。
		这会引起挤出效应，政府借大量的钱，企业会更难贷款买工厂和工具设施。

九、赤字与国债
	# 概念
		* 赤字，指的是某一年内，政府支出超出了税收，政府不得不借钱填补超出的支出。
		* 负债，是赤字的积累。
	# 美国的负债
		美国负债特别高，达到18万亿美元，全球第一。但是我们应该更加GDP情况来衡量美国的还款能力，美国的GDP也很高，因此【负债/GDP】并非特别高。
		从美国的赤字和税收来看，美国赤字的增加比税收增加的要快，也就是美国的负债一直在【积累】。
		* 高负债的原因
			政府支出的25%都花费在了社会保障上，25%花费在健康保健项目上，由于婴儿潮一代人的退休和长寿，美国老龄化加剧，因此在这上面花的钱就更多了。
	# 负债的缺点
		借钱需要债主，债主借钱给消费者，企业以及政府，政府借了太多的钱，债主的钱就会特别少，这时候债主可以借给消费者和企业的钱就更少了，并且利率上涨，导致经济衰退。
		希腊的巨量负债，导致了希腊政府无法还债，政府失去信用，债主大量损失金钱。
		
十、货币政策
	# 中央银行
		* 调控和监管国家的商业银行
		* 实施货币政策
			增加和减少货币供给，用于加速或放慢经济
		美国的中央银行，美国联邦储备系统，即美联储
		欧洲中央银行，ECB
	# 利率
		利率是借钱的成本。银行借出一定的钱，需要归还本金，以及以一定百分比弥补通货膨胀，创造收益，这个百分比是利率。
		利率低时，借款人容易还贷，于是就会借更多的钱，花更多的钱。反之不同。
		美联储可以通过改变货币，进而操控利率。
			美联储增加货币，会导致银行有更多的钱借出去，借钱人会货比三家，进而利率降低。更多借钱和消费，这是扩张性货币政策。
			美联储货币供给减少，会导致利率增加。紧缩性货币政策。
	# 货币供给的改变手段
		* 改变法定准备金
			银行存的钱，会留着一部分，另一部分会借出去。保留的这一部分就被称为【准备金】。
		* 改变控制银行的利率
			银行需要钱时，会到美联储借钱，美联储可以改变利率。美联储借给银行的利率被称为现贴率。
			现贴率越低，银行容易借钱，因此可以增加货币供给。
		* 公开市场操作
			美联储购买或增加短期【政府债券】。国家通过美联储向银行借钱，因此银行握有债券。
			美联储归还债券，就是增加货币供给。如果美联储发出更多的政府债券，则是减少货币供给。

十二、金融危机
	# 抵押贷款
		群众在银行借一大笔钱，用于购买某些产品，银行得到一张纸，群众每个月需要付一笔本金和利息给持有这张纸的单位，如果停止贷款，则持有这张纸的单位将会获得该产品，这就是抵押贷款。
		房贷按揭就是一种抵押贷款。
		持有这张纸的人不一定是银行，银行可能会把这张纸卖给第三方。
		抵押贷款需要好的信用和稳定的收入。
